# Pakistan COVID-19 Analysis Using SQL

## Project Overview

This project analyzes COVID-19 cases in Pakistan using SQL and MySQL Workbench to understand case trends, deaths, recoveries, travel history patterns, and province-level performance. The analysis transforms raw health data into meaningful business and public health insights through SQL queries and reporting techniques.

---

## Business Problem

During the COVID-19 pandemic, decision-makers needed accurate and timely information regarding case growth, recoveries, deaths, and regional distribution of infections. Understanding these trends is essential for resource planning, healthcare management, and policy decisions.

This project uses SQL to convert raw COVID-19 data into actionable insights.

## Business Questions

1. What are the total COVID-19 cases reported in Pakistan?
2. What are the total deaths and recoveries?
3. Which provinces have the highest number of cases?
4. Which provinces have the highest death rates?
5. Which cities report the highest number of cases?
6. What are the monthly COVID-19 trends?
7. What is the national recovery rate?
8. What is the national death rate?
9. How did travel history influence the spread of cases?
10. Which regions were most affected by COVID-19?

## Key Findings

* COVID-19 cases were concentrated in a few major provinces and cities.
* Recovery rates improved significantly over time.
* Death rates varied across provinces.
* Travel history contributed to early case distribution patterns.
* Monthly trends highlighted periods of rapid case growth.
